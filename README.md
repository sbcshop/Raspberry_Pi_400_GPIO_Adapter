# Raspberry Pi 400 GPIO Adapter

#### Raspberry Pi 400 GPIO Adapter is a new I/O expansion kit technological device built with 2 x 20 pins headers designed for Raspberry Pi 400. SB Components developed a Raspberry Pi GPIO header extension to provide a way to connect any type of HAT with Raspberry Pi 400.
